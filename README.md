# Smart Platform - UI - AngularJS
UI for Smart Platform in AngularJS
### Build
```
$ docker build -t smart-ui-angularjs .
```
### Start
```
$ docker run -d -p 8002:8002 smart-ui-angularjs
```
### Tools
- Public URLs for sending previews: https://ngrok.com/
